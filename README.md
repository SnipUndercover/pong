# Pong
A simple game of *Pong* in Svelte + Typescript.  
Built using Vite.  
Move the left paddle with `W`/`S`.  
Move the right paddle with `↑`/`↓`.  

## Setup
```bash
cd app
npm install
```

## Run
```bash
cd app
npm run dev
```

## Build
```bash
cd app
npm run preview # Preview app
npm run build   # Build app
```
